Translation memory (переводческая память, накопители переводов) – технология, позволяющая «не переводить одно и то же два раза». Программы для профессиональных переводов с данной технологией основываются на базах данных, которые содержат ранее переведенные единицы текста. Если в новом тексте обнаруживается единица, которая уже есть в базе, система автоматически добавляет ее в перевод. Такие программы значительно экономят время переводчика, особенно если он работает с однотипными текстами.

Технология перевода Translation Memory часто используется для решения таких задач, как перевод объёмных текстов разных стилей, перевод документов профессионального характера (на политические, экономические, социологические и прочие темы), а также перевод сайтов.

В интернете по данной теме можно найти историю появления технологии Translation Memory, информацию о её принципе работы, а также можно ознакомиться с конкретными программами, в которых используется эта технология.
